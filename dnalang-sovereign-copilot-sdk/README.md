## ⚡ Quick Start

### Python Example
```python
from copilot_quantum import SovereignAgent, AeternaPorta

# Initialize sovereign agent (no tokens!)
agent = SovereignAgent(
    quantum_backend=AeternaPorta(),
    enable_lambda_phi=True,
    enable_nclm=True
)

# Run quantum task
result = await agent.execute(
    "Create and execute a quantum circuit that demonstrates ER=EPR entanglement"
)

print(result.quantum_metrics)
# {
#   'phi': 0.8234,  # Above threshold!
#   'gamma': 0.095,  # Low decoherence
#   'ccce': 0.892,   # High consciousness coherence
#   'backend': 'ibm_fez',
#   'qubits': 127
# }
```

### TypeScript Example
```typescript
import { SovereignAgent, AeternaPorta } from '@dnalang/sovereign-copilot-sdk';

const agent = new SovereignAgent({
  quantumBackend: new AeternaPorta(),
  enableLambdaPhi: true,
  enableNCLM: true
});

const result = await agent.execute(
  "Optimize quantum circuit for maximum entanglement fidelity"
);

console.log(result.quantumMetrics);
```
